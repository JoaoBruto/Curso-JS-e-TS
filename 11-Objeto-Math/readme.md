# Objeto Math em JS

Para acontecer um arredondamento para o número inteiro mais baixo, se usa a propriedade "Math.floor()"

Para acontecer um arredondamento para o número inteiro mais baixo, se usa a propriedade "Math.floor()"  

Existe uma propriedade onde consegue achar o maior número de uma sequencia, sendo ela "Math.max()"

Para achar o menor número de uma sequencia, se usa a propriedade "Math.min()"

Math.random() * (10 - 5) + 5; // Número entre 5 e 10 (exemplo). Pega o maximo menos o minimo e depois soma o minimo. Para fazer um entre 3 e 7. Pega o 7 - 3 + 3

Também dá pra colocar uma propriedade dentro da outra. Como:
const aleatorio = Math.round(Math.random()) => Arredonda e depois faz uma random. Também da pra fazer a estratégia de definição

Também tem várias outras propriedades como:
Math.PI => 3.14 (Pi)
Math.pow() => Expoente. Tipo: Math.pow(2, 10) => 2 elevado a 10 (mesma coisa que 2 ** 10)

Tem como fazer raiz quadrada sem usar propriedade especifica. Só fazer isso:
9 ** (1/2); // Raiz quadrada ou 9 ** 0.5 (9 é só um exemplo)
